# 💻 C++ Fundamentals

Repositório dedicado ao estudo prático de C++, com foco em fundamentos, lógica de programação e construção de base sólida.

---

## 🎯 Objetivo

Desenvolver domínio em:

- Lógica de programação
- Estruturas fundamentais da linguagem
- Escrita e leitura de código
- Resolução de problemas
- Entendimento de memória e funcionamento interno

---

## 🧠 Metodologia

Este repositório segue uma abordagem prática inspirada no modelo da escola 42:

- Escrita contínua de código
- Repetição para ganho de fluência
- Aprendizado baseado em erro
- Estudo de padrões através de exemplos
- Evolução progressiva por níveis

Cada módulo contém:

- `examples/` → códigos de estudo e referência
- `exercises/` → problemas para prática e fixação
